# Traffic Accident Distance Analysis 
## Project Overview
This project focuses on **Exploratory Data Analysis (EDA)** and **Regression Modeling** of a large dataset containing over **146,000 US traffic accident records**. The main goal is to analyze factors affecting accident distance and severity using R.
## Key Objectives
* Clean and preprocess a large-scale dataset.
* Perform statistical analysis to identify patterns in traffic accidents.
* Build a regression model to understand the relationship between road conditions, weather, and accident impact.
## Tech Stack
* **Language:** R
* **Environment:** R Studio
* **Main Libraries:** `tidyverse`, `ggplot2`
## Project Structure
* `project_r_studio.R`: The complete R script containing data cleaning, visualization, and modeling.
* `SDS_REPORT.pdf`: A detailed final report with visualizations and statistical conclusions.
* `README.md`: Project documentation.
